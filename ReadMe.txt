WordsComplete.
This is a words completion plugin for FAR manager.

How to install?
Just like any FAR plugin. Put the DLL into the folder Far\Plugins\WordsComplete.

How to use?
Just type few first letter of a word and press Ctrl-Space or App Key, 
the plugin will search for words with the same beginning and show you a list.
